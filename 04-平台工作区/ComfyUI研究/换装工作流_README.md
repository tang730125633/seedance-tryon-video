# 云端 GPU 换装工作流

目标：把 `测试111换装测试/f2ebeb6f78e789aff274edc48391b8f9.mp4` 里的女性，换成 `图三.jpg` 的蓝白格套装，并把背景换成 `图二.jpg` 的粉色花艺场景。

## 推荐路线

### A. 先做稳定首帧

1. 从视频抽首帧或代表帧。
2. 用图三做衣服参考，图二做背景参考。
3. 通过 VTON / 图像编辑得到一张身份稳定的首帧。
4. 用首帧作为后续视频生成或视频重绘的强参考。

这个方式最快，适合先验证审美和方向。

### B. 云端 GPU 视频换装

1. AutoDL 开机，更新 `~/.autodl/instance.env`。
2. 在云端安装 ComfyUI + VTON 节点：

```bash
bash /root/autodl-tmp/work/cloud_vton_setup.sh
```

本项目里的安装脚本在：

```bash
90-AI脚本与代码/scripts/cloud_vton_setup.sh
```

3. ComfyUI 内建议组合：

- `VideoHelperSuite`：载入视频、拆帧、合成视频。
- `CatVTON` 或 `IDM-VTON`：对每帧或关键帧做衣服替换。
- `SAM2 / 人体分割`：把人物从原背景里抠出来。
- 背景合成：用图二做背景，裁成 9:16，匹配原视频 1088x1920。
- 一致性处理：优先关键帧换装，再用视频模型/ControlNet/光流约束扩展，避免逐帧闪烁。

## 当前素材判断

原视频是近景自拍，上半身和脸占画面很大，手会遮挡脸和衣服。图三是全身平铺衣服，因此视频里实际能稳定呈现的主要是：

- 蓝白格上衣
- 肩带/荷叶边
- 胸前白色蝴蝶结
- 前排纽扣

白色裙摆只会在下缘露一点，除非重新生成更远景构图。

## 风险

- 逐帧 VTON 容易闪烁。
- 手臂遮挡时，衣服边缘容易糊。
- 背景里的花枝和人物发丝/耳饰交界容易穿帮。
- 如果要保留原视频动作，不能只用普通图生视频，需要视频帧级处理或视频到视频模型。

## 最小可交付版本

1. 先出一张换装+换背景首帧。
2. 用 Seedance/Grok/XiaoLe 的图生视频延展为 5-8 秒。
3. 如果身份或动作不稳，再上 ComfyUI 云 GPU 逐帧/关键帧方案。
