Replaces the audio on an existing video and re-animates the speaker's lip movements to match the new audio. Use mode: 'speed' for fast output or 'precision' for high-quality lip-sync.

Usage:
  heygen lipsync create [flags]

Flags:
      --callback-id string          ID included in webhook payload
      --callback-url string         Webhook URL for completion notifications
  -d, --data string                 JSON request body (inline JSON, file path, or - for stdin). When used with individual flags, flags override matching fields in the JSON.
      --disable-music-track         Remove background music
      --enable-caption              Generate captions for the output video
      --enable-dynamic-duration     Allow dynamic duration adjustment (default true)
      --enable-speech-enhancement   Enhance speech quality
      --enable-watermark            Add watermark to output
      --end-time float              End time in seconds for partial lipsync
      --folder-id string            Project/folder ID to organize lipsync into
      --fps-mode string             Frame rate mode: 'vfr', 'cfr', or 'passthrough'.
  -h, --help                        help for create
      --keep-the-same-format        Preserve the source video's encoding specs (resolution, bitrate).
      --mode string                 Quality mode: 'speed' (faster) or 'precision' (higher quality, uses avatar inference) (allowed: speed, precision) (default "speed")
      --request-schema              Output the underlying API request body JSON schema and exit
      --response-schema             Output the underlying API response JSON schema and exit
      --start-time float            Start time in seconds for partial lipsync
      --timeout duration            Max time to wait when using --wait (default 20m0s)
      --title string                Title for the lipsync job
      --wait                        Poll until the operation completes or fails

Global Flags:
      --human   Display output as a formatted table instead of JSON

Examples:
# Create a lipsync job from JSON (use --request-schema for required fields)
  heygen lipsync create -d request.json
