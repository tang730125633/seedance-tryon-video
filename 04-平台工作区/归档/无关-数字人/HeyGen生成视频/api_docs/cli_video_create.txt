Creates a video from a HeyGen avatar or an arbitrary image. Supports scripts or pre-recorded audio for lip-sync. Supports Avatar IV and Avatar V engines; set the 'engine' field to select. Avatar III video generation requires the legacy API (v1 or v2).

Use --request-schema to see all available request fields.

Usage:
  heygen video create [flags]

Flags:
  -d, --data string        JSON request body (inline JSON, file path, or - for stdin). When used with individual flags, flags override matching fields in the JSON.
  -h, --help               help for create
      --request-schema     Output the underlying API request body JSON schema and exit
      --response-schema    Output the underlying API response JSON schema and exit
      --timeout duration   Max time to wait when using --wait (default 20m0s)
      --wait               Poll until the operation completes or fails

Global Flags:
      --human   Display output as a formatted table instead of JSON

Examples:
# Create an avatar video and wait for completion
  heygen video create -d '{"type":"avatar","avatar_id":"<avatar-id>","script":"Hello world","voice_id":"<voice-id>"}' --wait
# Create from a custom image
  heygen video create -d '{"type":"image","image":{"type":"url","url":"https://example.com/photo.jpg"},"script":"Hello","voice_id":"<voice-id>"}' --wait
# See all available request fields
  heygen video create --request-schema
