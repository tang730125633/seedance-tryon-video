# 定制化真人口播 · 技术方案（如何做到的 + 怎么快速调节）

> 目标：用任何人的「一张照片 + 一段录音」，生成他本人的、真实的、有表现力的口播视频。全脚本化、可批量、按量付费极便宜（一条30秒≈$0.2-0.6）。

## 🔑 万能配方（2026-06-20 验证，最重要的一条）
> **任意音频 + 一张清晰正脸图 → HeyGen `audio_asset_id` 对口型 → 真实口播**
>
> 音频来源随意组合，HeyGen 只负责把脸动起来精准对口型：
> - **本人真录音** → 最高保真"数字复刻"（声音=本人真情绪，零违和，最像）
> - **IndexTTS2 合成情绪声** → 可换任意新文案/批量/给别人用（声音稍逊真声）
> - **克隆别人的声音** → 谁的脸配谁的声
>
> = 把"真人口播"变成**可编程、可批量**的东西。音频换谁、说什么，完全自定。

### 🏆 实战验证：大鹏「医美新规大洗牌」一条龙复刻
1. whisper 扒原视频文案 → 清理存档（`口播文案库_大鹏_医美新规大洗牌.md`）
2. **底图用原视频怼脸截图**（t30帧，裁掉底部字幕条）→ 比影楼照更原生：场景/眼神/质感对味
3. 三种音频都跑通：
   - `大鹏_医美洗牌_原声数字复刻.mp4`（**本人真声**，109s，Tang评"非常牛逼"）
   - `大鹏_医美洗牌_完整版复刻.mp4`（IndexTTS2 合成激情声，92s）
4. HeyGen：`audio_asset_id`+原图，expressiveness=high，9:16，1080p

### ⚡ 新增关键经验
- **底图优先用"目标场景的真实截图"**（原视频帧/工作照），比影楼摆拍更自然、背景更可信。
- **IndexTTS2 情绪：`emo_text` 中文描述会被判成 calm！** 想要真激情必须用 **`emo_vector`** 直接喂8维向量 `[happy,angry,sad,afraid,disgusted,melancholic,surprised,calm]`。激情揭露感参考：`0.45,0.2,0,0,0,0,0.35,0` alpha 0.8。脚本已加 `--emovec`。
- **声音情绪 ↔ 脸表现力必须匹配**：声音激情就把 expressiveness=high；声音平和就用 medium。错位（激情声+平静脸）=违和（v1哭腔教训）。
- **长文案OK**：474字/92-109秒一次生成没问题，IndexTTS2 自动分段。

## 一、整体架构（数据流）
```
某人录音(10-60秒)  ──► HeyGen声音克隆 ──► voice_id（永久，可复用）
某人照片          ──► EXIF方向修正 ──► 上传asset ──► image asset_id
                                                          │
文案script + 表现力expressiveness + 肢体语言motion_prompt ─┤
                                                          ▼
                            HeyGen video create (Avatar IV)
                                          │
                                  轮询 ──► 下载（1080p无水印）
```
- **引擎**：HeyGen Avatar IV（照片动画，type=image，默认就是它）
- **接入**：官方 HeyGen CLI（`~/.local/bin/heygen`）+ API key（`~/.heygen/credentials`）
- **网络**：HeyGen 是美国服务器，**必须走 Mac 代理 `127.0.0.1:1082`**
- **计费**：API 钱包（独立于网页VIP），按视频分钟计，~$1-1.3/分钟

## 二、🎛️ 可调节的"定制旋钮"（这是核心，想调什么改什么）

| 旋钮 | 字段 | 取值/怎么调 | 影响 |
|---|---|---|---|
| **形象** | `image` | 换照片（清晰正脸，EXIF要修正） | 数字人长谁的脸 |
| **声音** | `voice_id` | 克隆某人 → 用他的voice_id | 数字人用谁的声音 |
| **⭐表现力** | `expressiveness` | **`high`** / medium / low（**默认low=太平!**） | 神态/情绪的丰富度（大鹏反馈的核心）|
| **肢体语言** | `motion_prompt` | 自然语言描述：双手抱胸/玩手指/前倾/手势… | 身体动作的自然度 |
| **文案** | `script` | 任意文本（HeyGen按voice_id念） | 说什么 |
| **语速/音调/音量** | `voice_settings` | speed 0.5-1.5 / pitch ±50 / volume | 声音微调 |
| **画质** | `resolution` | 4k / 1080p / 720p | 清晰度（720p便宜，1080p交付）|
| **比例** | `aspect_ratio` | 9:16(竖)/16:9/1:1/4:5… | 竖屏抖音用9:16 |
| **字幕** | `caption` | 对象（sidecar字幕，可能不烧进画面）| 字幕（建议后期ffmpeg烧关键词高亮）|

### 关键经验（大鹏案例验证）
- **"太平/人机"的主因 = `expressiveness` 默认 low** → 改成 **high** 立刻有情绪。这是大鹏老师亲口反馈"情绪最重要、别像死人"后定位到的。
- **克隆素材越长越自然**：13秒能克隆，但 **30-60秒自然带情绪的录音**克隆出来明显更准更有感情。
- **若 high 还不够"有情绪"** → 杀手锏见第五节（IndexTTS2 情绪控制）。

## 三、🚀 一键生成（以后调节就用这个）
脚本：`scripts/make_koubo.py`（自动修EXIF方向+上传+生成+轮询+下载）
```bash
python3 scripts/make_koubo.py \
  --photo "HeyGen生成视频/大鹏老师/大鹏_正脸_fixed.jpg" \
  --voice b859c12ff2964feeb818e5fdd802d417 \
  --script "美业单店越来越难做？……我是史立鹏，自由连锁，让每个美业老板都能赢" \
  --expressiveness high \
  --motion "自信热情演讲，手势配合强调，说到关键处身体前倾，自然眨眼，有气场" \
  --resolution 1080p --ratio 9:16 \
  --out "HeyGen生成视频/大鹏老师/大鹏_新版.mp4"
```
**想调什么就改对应参数**：换人→改 --photo + --voice；换内容→改 --script；更有情绪→--expressiveness high；换动作→--motion。

## 五、进阶：最强情绪控制（杀手锏，回应大鹏"情绪波动"）
HeyGen 的 expressiveness=high 已能提升，但若要**精确控制情绪**（兴奋/激动/沉稳/亲切，强度可调）：
- 用 **IndexTTS2（AutoDL GPU）零样本克隆** + `emo_text`/`emo_vector` 情绪参数生成音频
- 把这段**有情绪的音频**喂给 HeyGen（`audio_asset_id` 替代 script+voice_id）做对口型
- = 最强中文情绪声 + HeyGen真实对口型。脚本见 GPU 那套（`~/.autodl/idxtts.sh`）

## 六、踩过的坑（详见 api_docs/README_速查.md）
1. `engine` 字段 image 类型不能加 2. asset 上传统一返回 `data.asset_id` 3. m4a 要转 mp3 4. `video get` 是位置参数 5. **照片必须修EXIF方向**（手机/HEIF横拍会出歪脸）6. 必须走代理1082 7. **expressiveness 默认low**

## 七、成本参考
- 声音克隆：基本免费/极低
- 视频：~$1-1.3/分钟。一条30秒 ≈ $0.3-0.6。$15 钱包够做几十条。
